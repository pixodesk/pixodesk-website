# Add Lottie Animation to Astro Project
<SpacerBottom />

You can add Lottie animations to your Astro project using the @lottiefiles/dotlottie-web package.
<SpacerBottom size='1rem'/> 

It is a lightweight library that does not require you to install any other additional frameworks.
<SpacerBottom size='1rem'/> 

The steps to add a Lottie animation to a Astro project are: 
<SpacerBottom size='1rem'/> 
- Create an Astro project 
- Install dotLottie 
- Add your Lottie animation to the project 
- Create an animation component
- Use it in an Astro page
- Run your app 
- Edit the animation from the project folder (transform, colors, shapes, etc.) 
- Done! The animation is ready.

<SpacerBottom />
<SpacerBottom />
---

## Create Astro Project
<SpacerBottom />
Open your terminal and navigate to your projects folder:

```bash
cd path/to/your/projects
```

Create a new Astro project:

```bash
npm create astro@latest my-animation-app
```

Go into your project folder:

```bash
cd my-animation-app
```

---

## Install dotLottie

```bash
npm install @lottiefiles/dotlottie-web
```

---

## Add Your Lottie Animation
<SpacerBottom size='1rem' />

To add a Lottie animation to your project, you can:
<SpacerBottom size='1rem' />

- create it from scratch in [`Pixodesk 2D Animator`](https://pixodesk.com/2d-animator) or 
- download an animation from [`LottieFiles`](https://lottiefiles.com) and edit it in [`2D Animator`](https://pixodesk.com/2d-animator).
<SpacerBottom size='1rem' />

Create an `animations` folder in `public` and add your animation:

```bash
public/animations/animation.json
```

---

## Create Animation Component
<SpacerBottom size='1rem' />
**src/components/LottieAnimation.astro**



```astro
<div class="animation-container">
  <canvas id="dotlottie-canvas"></canvas>
</div>

<script>
  import { DotLottie } from '@lottiefiles/dotlottie-web';

  const canvas = document.getElementById('dotlottie-canvas') as HTMLCanvasElement;

  if (canvas) {
    new DotLottie({
      autoplay: true,
      loop: true,
      canvas,
      src: '/animations/animation.json',
    });
  }
</script>

<style>
  .animation-container {
    width: 300px;
    height: 300px;
  }

  canvas {
    width: 100%;
    height: 100%;
  }
</style>
```

## Use in Astro Page
<SpacerBottom size='1rem' />

**src/pages/index.astro**

```astro
---
import LottieAnimation from '../components/LottieAnimation.astro';
---

<html lang="en">
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width" />
        <title>Lottie Animation in Astro</title>
    </head>
    <body>
        <main>
            <h1>My Lottie Animation</h1>
            <LottieAnimation />
            {/* ... the rest of your code */}
        </main>
    </body>
</html>
```

## Run Your App
<SpacerBottom />

Start the development server:

```bash
npm run dev
```



You should see `Local: http://localhost:4321` in your terminal window. 

<SpacerBottom size='1rem' />

**Open the link in your browser.**
<SpacerBottom size='1rem' />
Note: the localhost port can be different if 4321 has been taken by another application.

<SpacerBottom size='1rem' />

Your Lottie animation should now work in your Astro project.

<SpacerBottom />


---

## Edit Animation From Project Folder
<SpacerBottom />
- Open your animation in **`2D Animator`** straight from your `public/animations` folder.
- **Modify** your file and **save** it.
