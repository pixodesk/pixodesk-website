# Add Lottie Animation to Next.js Project
<SpacerBottom />

You can add Lottie animations to a Next.js app using the `lottie-react` or `dotLottie-react` packages.
<SpacerBottom size='1rem'/> 
 
To add a Lottie animation to a Next.js project:
<SpacerBottom size='1rem'/> 

- Create a Next.js project 
- Install a Lottie player for React
- Add your Lottie animation to the project 
- Create a Lottie component 
- Use the component on the page
- Run your app 
- Edit the animation in the project folder (transform, colors, shapes, etc.)
- Done! The animation is ready

<SpacerBottom />
---

## Create Next.js Project
<SpacerBottom />

Open your terminal and navigate to your projects folder:

```bash
cd path/to/your/projects
```

Create a new Next.js project with TypeScript called `my-animation-app`:

```bash
npx create-next-app@latest my-animation-app --typescript
```


Go into your project folder:

```bash
cd my-animation-app
```

Then install dependencies:

```bash
npm install
```

<SpacerBottom size='1rem' />

---

## Install Lottie for Next.js
<SpacerBottom />

Install the `lottie-react` or `dotlottie-react` Lottie player.
<SpacerBottom size='1rem' />

**Option 1**: If you want to use `lottie-react`, in your Next.js project folder, run:

```bash
npm install lottie-react
```
<SpacerBottom size='1rem' />


**Option 2**: If you want to use `dotlottie-react`, in your Next.js project folder, run:

```bash
npm install @lottiefiles/dotlottie-react
```
---

## Add Your Lottie Animation
<SpacerBottom size='1rem' />

To add a Lottie animation to your project, you can:
<SpacerBottom size='1rem' />

- Create an animation from scratch in <span style="color: var(--sl-color-accent-high);">[`Pixodesk 2D Animator`](https://pixodesk.com/2d-animator)</span>, or 
- Download an animation from <span style="color: var(--sl-color-accent-high);">[`LottieFiles`](https://lottiefiles.com)</span> and edit it in <span style="color: var(--sl-color-accent-high);">[`2D Animator`](https://pixodesk.com/2d-animator)</span>.
<SpacerBottom size='1rem' />

Create an `animations` folder and add your animation:

```bash
app/animations/animation.json
```

---

## Create Lottie Component
<SpacerBottom />

Create a client component, since Lottie needs browser APIs.

How you use Lottie in Next.js depends on which player you are using to play the animation.
<SpacerBottom size='1rem' />

**Option 1**: If you are using `lottie-react`, **app/components/LottieAnimation.tsx** should look like this:
<SpacerBottom size='1rem' />


```tsx
"use client";

import Lottie from "lottie-react";
import animationData from "../animations/animation.json";

export default function LottieAnimation() {
    return (
        <div>
            <Lottie 
                animationData={animationData}
                loop
                autoplay
                style={{ width: 300, height: 300 }}
            />
            {/* ... the rest of your code */}
        </div>
    );
}

```
<SpacerBottom />


**Option 2**: If you are using `dotlottie-react`, **app/components/LottieAnimation.tsx** should look like this:
<SpacerBottom size='1rem' />

```tsx
"use client";

import { DotLottieReact } from '@lottiefiles/dotlottie-react';
import animationData from "../animations/animation.json";

export default function LottieAnimation() {
    return (
        <div>
            <DotLottieReact 
                data={animationData}
                loop
                autoplay
            />
            {/* ... the rest of your code */}
        </div>
    );
}

```
<SpacerBottom />


**Note:** Add **"use client"** at the top of components using Lottie, as it requires browser APIs.

<SpacerBottom />

## Use in Your Page
<SpacerBottom size='1rem' />
**app/page.tsx**

```tsx
import LottieAnimation from "./components/LottieAnimation";

export default function Home() {
    return (
        <div>
            <LottieAnimation />
            {/* ... the rest of your code */}
        </div>
    );
}
```

---

## Run Your App
<SpacerBottom />

Start the development server:

```bash
npm run dev
```


You should see `Local: http://localhost:3000` in your terminal window. 
<SpacerBottom size='1rem' />

**Open the link in your browser.**
<SpacerBottom size='1rem' />

**Note:** The localhost port may be different if `3000` is already being used by another application.
<SpacerBottom size='1rem' />

Your Lottie animation should now work in your Next.js project.
<SpacerBottom />



---

## Edit Animation in the Project Folder
<SpacerBottom />
- Open your animation in **`2D Animator`** directly from your `app/animations` folder.
- **Modify** your file and **save** it.
