# Add Lottie Animation to Nuxt Project
<SpacerBottom />

You can add Lottie animations to a Nuxt app using:
<SpacerBottom size='1rem'/> 

- the `nuxt-lottie` module 
- the `@lottiefiles/dotlottie-vue` package
<SpacerBottom />
---

## Option 1: Using the `Nuxt Lottie` Module
<SpacerBottom />

The Lottie module can be added to a Nuxt project as it is listed in the <span style="color: var(--sl-color-accent-high);">[`Nuxt Modules`](https://nuxt.com/modules/lottie)</span> directory.
<SpacerBottom size='1rem'/> 

To add a Lottie animation to a Nuxt app:
<SpacerBottom size='1rem'/> 

- Create a Nuxt project
- Add Nuxt Lottie module
- Add your Lottie animation
- Use Lottie in a Nuxt app
- Run your app
- Edit the animation in the project folder
- Done! The animation is ready
<SpacerBottom />

--- 

### Create Nuxt Project
<SpacerBottom />

Open your terminal and navigate to your projects folder:

```bash
cd path/to/your/projects
```

Create a new Nuxt project with TypeScript:

```bash
npx nuxi@latest init my-animation-app
```

Go into your project folder:

```bash
cd my-animation-app
```

Then install dependencies:

```bash
npm install
```


---


### Add Nuxt Lottie module
<SpacerBottom />

```bash
npx nuxi module add lottie
```
After installation, ensure that the `assets/lottie` folder exists. The module should automatically create it for you.
<SpacerBottom />


---

### Add Your Lottie Animation
<SpacerBottom size='1rem' />


To add a Lottie animation to your project, you can:
<SpacerBottom size='1rem' />

- Create an animation from scratch in <span style="color: var(--sl-color-accent-high);">[`Pixodesk 2D Animator`](https://pixodesk.com/2d-animator)</span>, or 
- Download an animation from <span style="color: var(--sl-color-accent-high);">[`LottieFiles`](https://lottiefiles.com)</span> and edit it in <span style="color: var(--sl-color-accent-high);">[`2D Animator`](https://pixodesk.com/2d-animator)</span>.
<SpacerBottom size='1rem' />



Add your animation to the `assets/lottie` folder.

```bash
assets/lottie/animation.json
```

---




### Use Lottie in Nuxt App
<SpacerBottom size='1rem' />

**app.vue**

```vue
<template>
    <div>
        <Lottie name="animation" width="300px" height="300px" />
        <!-- ... the rest of your code -->
    </div>
</template>

```

---



---

### Run Your App
<SpacerBottom />

Start the development server:
```bash
npm run dev
```

You should see `Local: http://localhost:3000` in your terminal window. 
<SpacerBottom size='1rem' />

**Open the link in your browser.**
<SpacerBottom size='1rem' />

**Note:** The localhost port can be different if `3000` has been taken by another application.
<SpacerBottom size='1rem' />

Your Lottie animation should now work in your Nuxt app.
<SpacerBottom />



---

### Edit Animation in the Project Folder
<SpacerBottom />

- Open your animation in **`2D Animator`** directly from your `assets/lottie` folder.
- **Modify** your file and **save** it. 
<SpacerBottom />

---

<SpacerBottom />

## Option 2: Using the `@lottiefiles/dotlottie-vue` Package
<SpacerBottom />

You can add Lottie animations to a Nuxt app using the `@lottiefiles/dotlottie-vue` package.
<SpacerBottom size='1rem'/> 

To add a Lottie animation to a Nuxt app:
<SpacerBottom size='1rem'/> 

- Create a Nuxt project 
- Install the Lottie player for Vue
- Add your Lottie animation to the project 
- Create a Lottie component 
- Use Lottie in a Nuxt app
- Run your app
- Edit the animation in the project folder (transform, colors, shapes, etc.)
- Done! The animation is ready
<SpacerBottom />


---





### Create Nuxt Project

Open your terminal and navigate to your projects folder:

```bash
cd path/to/your/projects
```

Create a new Nuxt project with TypeScript:

```bash
npx nuxi@latest init my-animation-app
```

Go into your project folder:

```bash
cd my-animation-app
```

Then install dependencies:

```bash
npm install
```


---


### Install Lottie for Vue

```bash
npm install @lottiefiles/dotlottie-vue
```
---

### Add Your Lottie Animation
<SpacerBottom size='1rem' />


To add a Lottie animation to your project, you can:
<SpacerBottom size='1rem' />

- Create an animation from scratch in <span style="color: var(--sl-color-accent-high);">[`Pixodesk 2D Animator`](https://pixodesk.com/2d-animator)</span>, or 
- Download an animation from <span style="color: var(--sl-color-accent-high);">[`LottieFiles`](https://lottiefiles.com)</span> and edit it in <span style="color: var(--sl-color-accent-high);">[`2D Animator`](https://pixodesk.com/2d-animator)</span>.
<SpacerBottom size='1rem' />



Create an `animations` folder in `public` and add your animation:

```bash
public/animations/animation.json
```

---

### Create Lottie Component
<SpacerBottom />

Next to the app.vue file, create a `components` folder and add LottieAnimation.vue to it.

<SpacerBottom size='1rem' />
**components/LottieAnimation.vue**

```vue
<template>
    <div class="lottie-container">
        <DotLottieVue
            src="/animations/animation.json"
            :loop="true"
            :autoplay="true"
            style="height: 300px; width: 300px;"
        />
    </div>
</template>

<script setup lang="ts">
import { DotLottieVue } from '@lottiefiles/dotlottie-vue';
</script>

<style scoped>
.lottie-container {
    display: flex;
    justify-content: center;
    align-items: center;
    width: 100%;
}
</style>
```

---

### Use Lottie in Nuxt App
<SpacerBottom size='1rem' />

**app.vue**

```vue
<template>
    <div>
        <div class="animation">
            <LottieAnimation />
        </div>
        <!-- ... the rest of your code -->
    </div>
</template>


<style scoped>
.animation {
    display: flex;
    flex-direction: column;
    align-items: center;
    justify-content: center;
}
/* ... the rest of your style */
</style>


```

---

### Run Your App
<SpacerBottom />

Start the development server:
```bash
npm run dev
```

You should see `Local: http://localhost:3000` in your terminal window. 
<SpacerBottom size='1rem' />

**Open the link in your browser.**
<SpacerBottom size='1rem' />

**Note:** The localhost port can be different if `3000` has been taken by another application.
<SpacerBottom size='1rem' />

Your Lottie animation should now work in your Nuxt app.
<SpacerBottom />



---

### Edit Animation in the Project Folder
<SpacerBottom />

- Open your animation in **`2D Animator`** directly from your `public/animations` folder.
- **Modify** your file and **save** it. 
<SpacerBottom />

---

<SpacerBottom />
