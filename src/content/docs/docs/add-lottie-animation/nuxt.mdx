# Add Lottie Animation to Nuxt Project
<SpacerBottom />
You can add Lottie animations to a Nuxt app easily using the `@lottiefiles/dotlottie-vue` package.
<SpacerBottom />
---

## Create Nuxt Project
<SpacerBottom />
Open terminal and navigate to your projects folder:

```bash
cd path/to/your/projects
```

Create a new Nuxt project with TypeScript:

```bash
npx nuxi@latest init my-animation-app
```
Enter your project directory:
```bash
cd my-animation-app
```

Install dependencies:

```bash
npm install
```


---

## Install Lottie Package

```bash
npm install @lottiefiles/dotlottie-vue
```
---

## Get Your Lottie Animation
<SpacerBottom size='1rem' />
### Create an animation in 2D Animator
- <a href="https://pixodesk.com/2d-animator" target="_blank">https://pixodesk.com/2d-animator</a><Badge text="Tool" variant="note" />

### Download from LottieFiles
- <a href="https://lottiefiles.com" target="_blank">https://lottiefiles.com</a>

<SpacerBottom size='1rem' />

In public folder create an `animations` folder and add your animation:

```bash
public/animations/animation.json
```

---

## Create Lottie Component
<SpacerBottom />
Next to app.vue file create a components folder and add there LottieAnimation.vue. 
<SpacerBottom size='1rem' />
**components/LottieAnimation.vue**

```vue
<template>
    <div class="lottie-container">
        <DotLottieVue
        src="/animations/animation.json"
        :loop="true"
        :autoplay="true"
        style="height: 300px; width: 300px;"
        />
    </div>
</template>

<script setup lang="ts">
import { DotLottieVue } from '@lottiefiles/dotlottie-vue';
</script>

<style scoped>
.lottie-container {
    display: flex;
    justify-content: center;
    align-items: center;
    width: 100%;
}
</style>
```

---

## Use in Nuxt App
<SpacerBottom size='1rem' />
**app.vue**

```vue
<template>
  <div>
    <div class="animation">
      <LottieAnimation />
    </div>
    <!-- ... the rest of your code -->
    
  </div>
</template>

<script setup>
import LottieAnimation from './components/LottieAnimation.vue';
// ... the rest of your import
</script>

<style scoped>
.animation {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
}
/* ... the rest of your style */
</style>
```

---

<SpacerBottom />

Start the development server:
```bash

```

You should see:
```bash
 Local: http://localhost:3000
 ```
