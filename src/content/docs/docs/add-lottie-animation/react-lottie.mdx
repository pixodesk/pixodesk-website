# Add Lottie Animation to React Project
<SpacerBottom />
You can add Lottie animations to a React app very easily. The most common and recommended way is to use the lottie-react package.

This is a clean, modern, step-by-step approach.
<SpacerBottom />
---

## Create React Project

```bash
cd path/to/your/projects
```
Then:

```bash
npm create vite@latest my-animation-app -- --template react-ts
```

This creates a React + TypeScript project called my-animation-app.
<SpacerBottom size='1rem' />
Enter your project directory: 
```bash
cd my-animation-app
```
Install dependencies:

```bash
npm install
```

<SpacerBottom size='1rem' />
---

## Install Lottie for React
<SpacerBottom />
In your React project folder, run:

```bash
npm install lottie-react
```
<SpacerBottom size='1rem' />
---

## Get Your Lottie Animation
<SpacerBottom size='1rem' />
### Create an animation in 2D Animator
- <a href="https://pixodesk.com/2d-animator" target="_blank">https://pixodesk.com/2d-animator</a><Badge text="Tool" variant="note" />

### Download from LottieFiles
- <a href="https://lottiefiles.com" target="_blank">https://lottiefiles.com</a>
<SpacerBottom size='1rem' />

Create an `animations` folder in `src/assets/` and add your animation:

```bash
src/assets/animations/animation.json
```
<SpacerBottom size='1rem' />
---

## Use Lottie in React Component
<SpacerBottom size='1rem' />
**src/App.tsx**
```tsx
import Lottie from "lottie-react";
import animationData from "./assets/animations/animation.json";


function App() {
    return (
        <div>
            <Lottie 
                animationData={animationData}
                loop={true}
                autoplay={true}
                style={{ width: 300, height: 300 }}
            />
            {/* ... the rest of your code */}
        </div>
    );
}

export default App;

```
<SpacerBottom size='1rem' />
---

## Start Development Server

```bash
npm run dev
```
