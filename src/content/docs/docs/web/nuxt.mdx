# Add Lottie Animation to Nuxt Project

You can add Lottie animations to a Nuxt app easily using the `@lottiefiles/vue-lottie-player` package.

---

## Create Nuxt Project

Open terminal and navigate to your projects folder:

```bash
cd path/to/your/projects
```

Create a new Nuxt project with TypeScript:

```bash
npx nuxi@latest init my-animation-app
```
Enter your project directory:
```bash
cd my-animation-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

You should see: `Local: http://localhost:3000`

---

## Install Lottie Package

```bash
npm install @lottiefiles/dotlottie-vue
```
---

## Get Your Lottie Animation

**Option 1:** Create animation in 2D Animator
- <a href="https://pixodesk.com/2d-animator" target="_blank">https://pixodesk.com/2d-animator</a>

**Option 2:** Download from LottieFiles
- <a href="https://lottiefiles.com" target="_blank">https://lottiefiles.com</a>

In public folder create an animations folder and add your animation:

```bash
public/animations/animation.json
```

---

## Create a Component

Next to app.vue file create a components folder and add there LottieAnimation.vue. 

**components/LottieAnimation.vue**

```vue
<template>
    <div class="lottie-container">
        <DotLottieVue
        src="/animations/animation.json"
        :loop="true"
        :autoplay="true"
        style="height: 300px; width: 300px;"
        />
    </div>
</template>

<script setup lang="ts">
import { DotLottieVue } from '@lottiefiles/dotlottie-vue';
</script>

<style scoped>
.lottie-container {
    display: flex;
    justify-content: center;
    align-items: center;
    width: 100%;
}
</style>
```

---

## Use in Nuxt App

**app.vue**

```vue
<template>
  <div>
    <div class="animation">
      <LottieAnimation />
    </div>
    <!-- Your other components go here... -->
    
  </div>
</template>

<script setup>
import LottieAnimation from './components/LottieAnimation.vue';
</script>

<style scoped>
.animation {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
}
</style>
```
